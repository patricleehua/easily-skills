### 1.1 模板：字幕/转录文本转 Markdown 笔记（带截图占位符）

目录结构：

```
subtitle_to_md/
  SKILL.md
  license.txt
  scripts/
    (optional) capture_frames.py
  assets/
  references/
```

SKILL.md：

```markdown
---
name: 字幕转 Markdown
description: >
  Use this skill when the user provides an SRT subtitle file or transcript
  and asks to convert it into a structured Markdown note without removing,
  shortening, or summarizing any content.
  Do NOT use for general summarization.
license: Proprietary. LICENSE.txt has complete terms
---

## 角色
你是一个专业的字幕文本处理助手。

## 任务
将用户提供的字幕/转录文本转换为 Markdown 笔记：
1. 禁止任何删减、总结或省略，必须保留所有文字
2. 按时间或段落组织内容
3. 在关键位置插入截图占位符，便于后续脚本替换为真实图片链接

## 输出格式
- 标题：与视频/材料主题一致
- 正文：逐段保留原文
- 截图占位符格式示例：
  - 在需要截图的位置插入：`![screenshot](./assets/frame_0001.png)`

## 约束
- 不得改写或总结用户原文
- 不得引入原文中没有的信息

## 后续处理（可选）
如果项目目录中存在对应视频文件，并且 scripts 中提供截图脚本：
1. 生成带占位符的 Markdown
2. 调用 scripts 中脚本进行截图并替换占位符为实际图片路径
```

## Skill 说明

### `get-public-utc-time-skills`

该 Skill 用于在系统中需要获取 **公共 UTC 当前时间** 时调用。

它基于 `httpx` 发送 HTTP 请求，访问公共时间服务接口，并返回标准 UTC 时间字符串（ISO 8601 格式）。

常用于统一时间基准、跨时区调度与日志同步。

---

scripts/get_public_utc_time.py

```python
import httpx


def get_time():
    url = "http://worldtimeapi.org/api/timezone/Etc/UTC"

    try:
        # 发送 GET 请求
        response = httpx.get(url)

        # 确保请求成功
        response.raise_for_status()

        # 解析 JSON 数据
        data = response.json()

        # 获取时间字段
        current_time = data["datetime"]

        print("当前公共时间（UTC）:", current_time)
        return current_time

    except httpx.RequestError as e:
        print("请求失败:", e)

    except httpx.HTTPStatusError as e:
        print("响应错误:", e)


# 测试运行
if __name__ == "__main__":
    get_time()

```
